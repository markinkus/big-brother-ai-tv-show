# Persona Card System

Persona cards are structured fictional contestant blueprints generated before contestant creation.

MVP first-deliverable flow:

1. request deterministic candidate generation
2. persist generated cards for a season
3. review card details in the web UI
4. approve a card
5. create a contestant from the approved card

Safety posture:

- respectful fictional framing
- no hateful stereotypes
- no degrading depictions of protected groups
- religious and cultural references are sensitivity-aware and auditable
